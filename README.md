# Customer_dataAnalysis

## Project Overview
This project analyzes customer shopping behavior using transactional data from 3,900 purchases across multiple product categories. The objective is to uncover insights related to customer spending patterns, subscription trends, product performance, and purchasing behavior to support data-driven business decisions.

The project combines Python for data cleaning and exploratory data analysis, PostgreSQL for business query analysis, and Power BI for interactive dashboard visualization.

---

## Objectives
- Analyze customer purchase behavior and spending trends
- Identify high-performing products and categories
- Study the impact of discounts and subscriptions on revenue
- Segment customers based on purchase history
- Visualize insights using an interactive Power BI dashboard
